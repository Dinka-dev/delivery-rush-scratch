# delivery-rush-scratch

Delivery Rush é um mini-jogo desenvolvido em Scratch como meu primeiro projeto prático de programação, criado durante meus estudos no curso CS50.

**Sobre o jogo**

O jogador controla um entregador que deve coletar pedidos dentro de um tempo limite, acumulando pontos até o fim do jogo.

**Conceitos aplicados**

Lógica de programação

Variáveis e controle de estado

Loops (repeat until, forever)

Condições (if)

Eventos

Detecção de colisão

Controle de tempo

Debug e organização de scripts

Jogue aqui: https://scratch.mit.edu/projects/1271151351/

**Contexto**

Este projeto marca minha transição da teoria para a prática e faz parte do início da minha migração de carreira para a área de desenvolvimento de software.
